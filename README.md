# DigitalProduction.Project
A library for creating complex projects. It exposes a `Project` base class that contains common features for projects including:
- Serialization Features
    - As a single file
    - As multiple files in a zip archive
    - Binary or text based serialization
- Notifications of property changes
- Modified status and notification events
- Standard observable properties that expose if the project is saved, open, has a path, et cetera
